#deskripsi projek perentalan mobil 
Proyek Program Perentalan Mobil merupakan aplikasi berbasis bahasa pemrograman Python yang dirancang untuk mengelola proses penyewaan mobil secara sederhana dengan menerapkan konsep Object Oriented Programming (OOP) dan Function. Program ini berjalan pada terminal/console dan memungkinkan pengguna melakukan input data penyewaan serta menampilkan data rental yang telah tersimpan.

#yang di butuhkan oleh projek Perentalan Mobil ini

1. Perangkat Lunak
		Python versi 3.x
		Sistem Operasi (Windows / Linux / macOS)
		Text Editor atau IDE (VS Code, PyCharm, IDLE)

2. Perangkat Keras
		Laptop atau komputer
		Processor minimal Dual-Core
		RAM minimal 2 GB

3. Kebutuhan Sistem
		Menu input sewa mobil
		Penyimpanan data pelanggan
		Penyimpanan data mobil
		Penyimpanan data transaksi sewa
		Tampilan data rental mobil
		Status mobil (Tersedia / Disewa)

4. Kebutuhan Pengguna
		Admin / petugas rental
		Mengerti penggunaan komputer dasar

#bagaimana mengunakan projek ini
1. Lingkungan Pemrograman
Bahasa pemrograman Python (karena kode ditulis dalam Python)​
Editor/IDE untuk menulis dan menjalankan kode (misalnya VS Code, PyCharm, IDLE bawaan Python)
 
2. Struktur Dasar yang Sudah Ada tapi Perlu Koreksi
Perbaikan pada metode konstruktor: Di Python, konstruktor harus ditulis sebagai  _init_  (dua underscore di awal dan akhir), bukan  init  (satu underscore). Kesalahan ini akan menyebabkan kode tidak berjalan dengan benar.​
Class Induk dan Turunan: Class  Orang  sebagai induk sudah ada, namun perlu pastikan pewarisan ( Inheritance ) pada  Pelanggan  berfungsi setelah memperbaiki konstruktor.
 
3. Fitur yang Sudah Diimplementasikan 
 Input data pelanggan (nama, no HP) dan mobil (merk, plat) serta lama sewa.​
 Perubahan status mobil (Tersedia/Disewa) saat proses sewa.​
 Penyimpanan data sewa dalam list.​
 Tampilan data rental yang tersimpan.​
 Menu interaktif untuk pengguna.
  
4. Fitur Tambahan yang Bisa Dipertimbangkan (Opsional tapi Disarankan untuk Kelengkapan)
 Validasi input (misalnya no HP hanya angka, lama sewa tidak negatif, plat mobil sesuai format).​
 Manajemen daftar mobil yang sudah ada (bukan membuat mobil baru setiap kali input sewa).​
 Perhitungan biaya sewa berdasarkan merk mobil dan lama sewa.​
 Pencatatan tanggal sewa dan tanggal kembali.​
 Fungsi untuk mengembalikan mobil dan memperbarui statusnya.​
 Penyimpanan data ke file (misalnya CSV, JSON) agar tidak hilang saat program ditutup.
